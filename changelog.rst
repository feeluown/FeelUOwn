""""""""""""""""""""""

  - 给重构的代码添加单元测试，覆盖率从 45% 提至 56%。

- 不兼容改动

  - 不再支持 Python 3.6 和 3.7。
  - rcfile 文件中通过 when 函数绑定的信号接受者，默认不再放在队列中执行，
    而是直接调用。可以通过指定 `aioqueue=False` 来保持向后兼容。举个例子：
    `when('xxx.yyy', func, aioqueue=False)` 。
