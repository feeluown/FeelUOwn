# Simple NetEaseMusic for linux 

**项目主页**: <http://cosven.com/blog/5518934e126ac921c1f77a72>

> 简单的网易云音乐Linux客户端


## 依賴
Python 2.7, PyQt4, PyQt4-phonon

**Ubuntu下**

```
sudo apt-get install python-qt4
sudo apt-get install python-qt4-phonon
```
