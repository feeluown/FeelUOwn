媒体资源管理
===================

feeluown 一个设计目标是让用户能够高效使用各个音乐平台的合法资源。
媒体资源管理定义、规范并统一了各个音乐平台资源的访问接口。

音乐库是媒体资源管理子系统的入口。音乐库部分负责管理 feeluown 的音乐资源，
包括歌曲、歌手、专辑详情获取，专辑、歌单封面获取等。它主要由几个部分组成：
音乐对象模型(*Model*)、音乐提供方(*Provider*)、提供方管理(*Library*)。
