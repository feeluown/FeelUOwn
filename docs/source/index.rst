FeelUOwn - feel your own
========================

FeelUOwn 是一个用户友好、可玩性强的播放器

.. image:: https://user-images.githubusercontent.com/4962134/43506587-1c56c960-959d-11e8-964f-016159cbeeb9.png

它主要有以下特性：

- 安装简单，使用方便，新手友好
- 默认提供国内各音乐平台插件（网易云、虾米、QQ）
- 与 tmux，Emacs，Vim 等工具可以方便集成
- 核心模块有较好文档和测试覆盖
